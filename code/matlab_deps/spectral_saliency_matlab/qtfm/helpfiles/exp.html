Quaternion Function Reference

exp

Exponential
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = exp(X)

Description

exp(X) computes the exponential of its argument.

Examples

>> show(exp([qi, qj, qk]))
 
S = 0.5403    0.5403    0.5403

X = 0.8415         0         0

Y =      0    0.8415         0

Z =      0         0    0.8415
