Quaternion Function Reference

qr

QR decomposition
(Quaternion overloading of standard MATLAB® function)

Syntax

[Q,R] = qr(A)

Description

Computes a decomposition such that A = Q * R, where Q is unitary and R is upper triangular respectively.

See Also

MATLAB® function: qr
QTFM functions: lu, svd
