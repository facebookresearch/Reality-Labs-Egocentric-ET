Quaternion Function Reference

dc

Cayley-Dickson recomposition

Syntax

q = dc(A, B)

Description

dc is the 'inverse' of the function cd. Given two complex numbers it returns the quaternion whose Cayley-Dickson decomposition would yield the two complex numbers.
