Quaternion Function Reference

floor

Round towards minus infinity.
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = floor(X)

Description

floor(X) returns the quaternion resulting from applying the MATLAB® floor function to the components of the quaternion argument.
