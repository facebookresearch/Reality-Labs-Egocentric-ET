Quaternion Function Reference

qtfm_test

Run QTFM test code

Syntax

qtfm_test

Description

qtfm_test runs the test code supplied with QTFM. The test code may also be run using the MATLAB® Start menu. It is intended primarily for internal use, but may be used to run the test code from the command window.
