Quaternion Function Reference

vector_product

Vector or cross product of two pure quaternions

Syntax

Y = vector_product(A, B)

Description

vector_product(A, B) returns the vector (or cross) product of the two arguments, which must be pure quaternions.

Examples

The vector product of qi with qj gives qk: 
>> vector_product(qi, qj) == qk

ans = 1
