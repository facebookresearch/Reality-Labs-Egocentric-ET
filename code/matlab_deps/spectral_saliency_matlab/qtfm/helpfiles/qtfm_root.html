Quaternion Function Reference

qtfm_root

Path to QTFM installation folder or directory

Syntax

S = qtfm_root

Description

qtfm_root returns a string containing the full path to the root folder or directory where QTFM is installed. It is intended primarily for internal use, for example, in running test code from the MATLAB® Start menu.
