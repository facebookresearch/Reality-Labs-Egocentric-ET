Quaternion Function Reference

zerosv

Pure quaternion zeros matrix

Syntax

Y = zerosv(N)

Description

zerosv(X) returns a pure quaternion matrix of zeros. Accepts the same parameters as the MATLAB® zeros function.

Examples

>>  show(zerosv(2))
 
X =
 
     0     0
     0     0

Y =
 
     0     0
     0     0

Z =
 
     0     0
     0     0
