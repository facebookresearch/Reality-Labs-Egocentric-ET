Quaternion Function Reference

displayall

Display elements of array

Syntax

displayall(X)

Description

displayall(X) displays the value of the elements of quaternion X preceded by its name. show is a synonym for displayall.

Examples

>> A = randq(2)
 
A = 2x2 quaternion array
 
>> displayall(A)
 
A.S =
 
   -0.4697   -0.4876
    0.4331    0.1549

A.X =
 
   -0.3172   -0.1382
    0.2339    0.7641

A.Y =
 
    0.8121    0.4941
    0.2490   -0.5607

A.Z =
 
   -0.1387    0.7064
    0.8341    0.2789
