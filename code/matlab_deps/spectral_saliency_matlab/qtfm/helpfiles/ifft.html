Quaternion Function Reference

ifft

Inverse Fast Fourier transform
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = ifft(X)

Description

ifft(X) computes the inverse Fast Fourier Transform of the quaternion vector X using a default axis and left-handed transform. See the related function iqfft which permits control over the axis and handedness.

The axis chosen is a complex (biquaternion) axis if X is complex, otherwise the axis is a (real) quaternion.

The N and dim parameters of the overloaded MATLAB® function are not yet implemented, but may be implemented at a later date.

See Also

MATLAB® function: ifft
QTFM functions: ifft2, fft, iqfft
