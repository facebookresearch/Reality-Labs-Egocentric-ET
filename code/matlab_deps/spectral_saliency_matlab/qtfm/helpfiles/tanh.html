Quaternion Function Reference

tanh

Hyperbolic tangent
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = tanh(X)

Description

tanh(X) computes the hyperbolic tangent of the quaternion argument X which may be a complexified quaternion.
