Quaternion Function Reference

sin

Sine
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = sin(X)

Description

sin(X) computes the sine of the quaternion argument X which may be a complexified quaternion.

Examples

>>  sin(1 + qi)
 
ans = 1.298 + 0.635 * I + 0 * J + 0 * K
