Quaternion Function Reference

length

Length of vector
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = length(X)

Description

length(X) returns the number of quaternion elements in the vector X.

Examples

>> q = randq(7,1)
 
q = 7x1 quaternion array
 
>> length(q)

ans = 7
