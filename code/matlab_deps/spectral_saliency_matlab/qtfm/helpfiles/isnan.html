Quaternion Function Reference

isnan

Array elements containing NaNs
(Quaternion overloading of standard MATLAB® function)

Syntax

TF = isnan(X)

Description

isnan(X) returns logical 1 (true) for array elements that contain at least one quaternion component which is NaN.
