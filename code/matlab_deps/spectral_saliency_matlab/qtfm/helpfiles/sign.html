Quaternion Function Reference

sign

Signum function
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = sign(X)

Description

sign(X) is equivalent to unit(X). It is implemented so that MATLAB® code which calls sign will work for quaternion arrays.

Examples

>> sign(quaternion(1,2,3,4))
 
ans = 0.1826 + 0.3651 * I + 0.5477 * J + 0.7303 * K
