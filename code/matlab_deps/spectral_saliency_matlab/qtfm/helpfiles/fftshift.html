Quaternion Function Reference

fftshift

Shift zero-frequency component to center of spectrum.
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = fftshift(X, dim)

Description

fftshift overloads the MATLAB® function of the same name and implements the same functionality, for quaternion arrays.
