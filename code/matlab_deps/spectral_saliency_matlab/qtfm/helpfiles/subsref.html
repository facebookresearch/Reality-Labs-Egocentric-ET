Quaternion Function Reference

subsref

Subscripted reference
(Quaternion overloading of standard MATLAB® function)

Syntax

r = subsref(a, ss)

Description

subsref is an overloading of a MATLAB® function that plays an essential role in subscripted references. See the documentation on the MATLAB® function for details, and the source code of the quaternion implementation.
