Quaternion Function Reference

show

Display elements of array

Syntax

show(X)

Description

show(X) displays the value of the elements of quaternion X preceded by its name. displayall is a synonym for show.

Examples

>> show(eyeq(2))
 
S =
 
     1     0
     0     1

X =
 
     0     0
     0     0

Y =
 
     0     0
     0     0

Z =
 
     0     0
     0     0
