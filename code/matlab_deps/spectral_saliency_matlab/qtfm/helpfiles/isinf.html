Quaternion Function Reference

isinf

Array elements that are infinite
(Quaternion overloading of standard MATLAB® function)

Syntax

TF = isinf(X)

Description

isinf(X) returns logical 1 (true) for array elements that contain at least one quaternion component which is infinite. It returns logical 0 (false) at array positions where all quaternion components are finite.
