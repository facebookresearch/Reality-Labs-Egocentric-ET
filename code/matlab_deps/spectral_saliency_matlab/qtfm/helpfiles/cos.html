Quaternion Function Reference

cos

Cosine
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = cos(X)

Description

cos(X) computes the cosine of the quaternion argument X which may be a complexified quaternion.

Examples

>> cos(1 + qi)
 
ans = 0.8337 - 0.9889 * I + 0 * J + 0 * K
