Quaternion Function Reference

fix

Round towards zero.
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = fix(X)

Description

fix(X) returns the quaternion resulting from applying the MATLAB® fix function to the components of the quaternion argument.
