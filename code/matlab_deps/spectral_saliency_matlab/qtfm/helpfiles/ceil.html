Quaternion Function Reference

ceil

Round towards plus infinity.
(Quaternion overloading of standard MATLAB® function)

Syntax

Y = ceil(X)

Description

ceil(X) returns the quaternion resulting from applying the MATLAB® ceil function to the components of the quaternion argument.
