Quaternion Function Reference

subsasgn

Subscripted assignment
(Quaternion overloading of standard MATLAB® function)

Syntax

r = subsasgn(a, ss, b)

Description

subsasgn is an overloading of a MATLAB® function that plays an essential role in subscripted assignment. See the documentation on the MATLAB® function for details, and the source code of the quaternion implementation.
